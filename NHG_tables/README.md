# NHG tables

NHG tabel 4, 5, 6, 9, 12, 20, 24 were downloaded from [here](https://referentiemodel.nhg.org/tabellen/inkijkexemplaren) on 29/05/2024 as pdf format and converted to csv using Excel. Conversions "should" have been be accurate & complete, but haven't been checked extensively.

NHG tabel 45 was scraped from [here](https://bepalingen.nhg.org/labcodes/determinations) on 29/05/2024. Scraping was done in R, ask j.w.a.van_egeraat@lumc.nl for the code if you are interested or want to scrape a new version. Table should also be accurate & complete.
